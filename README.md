# Java Web 学习项目

## 项目概述

这是一个用于学习 Java Web 开发的实践项目集合，包含多个不同的演示应用。

## 更新记录

### 2025-11-24

初始版本创建，包含以下项目：
1. TianNanDemo - 天南大陆智能学习辅助系统
2. newsDemo - 新闻展示演示项目

## 项目详情

### TianNanDemo - 天南大陆智能学习辅助系统

一个模拟的学习管理系统界面，展示了现代化的前端设计技术。

主要功能：
- 响应式固定导航栏
- 员工信息搜索表单
- 数据表格展示
- 固定页脚信息栏

技术特点：
- 使用 HTML5 和 CSS3
- Flexbox 布局实现响应式设计
- CSS 定位技术（fixed 和 sticky）
- 语义化标签使用

### newsDemo - 新闻展示演示

一个简单的新闻页面模板，演示了多媒体内容的展示。

主要功能：
- 新闻标题和时间显示
- 视频播放器集成
- 图片内容展示
- 文本格式化排版

技术特点：
- HTML 基础标签使用
- CSS 样式控制
- 媒体元素嵌入 (video, img)
- 外部样式表引入

## 如何运行

1. 克隆或下载整个项目到本地
2. 使用浏览器直接打开相应的 HTML 文件即可查看效果
